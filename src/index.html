Contribute to the Project

The Magento PWA Studio project is an open source project that welcomes community contributions and collaboration.

All the tools listed below are available as separate packages in the PWA Studio repository.

If you have any questions, please join the #pwa Slack channel at magentocommeng.slack.com.

Buildpack

Build and layout optimization tooling for Magento PWA projects.

Peregrine

A collection of UI components for the Magento PWA projects.
